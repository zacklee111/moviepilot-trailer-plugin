# MoviePilot 预告片自动下载插件

电影入库后自动从 YouTube 下载预告片，保存为 `-trailer.mp4` 格式。

## 功能特点

- 🎬 电影入库自动触发下载
- 🌐 支持中文/英文预告片
- 📁 与正片保存在同一文件夹
- ⏭️ 跳过已存在的预告片
- 🎯 支持指定监控路径
- ⏰ 支持定时全库扫描

## 安装方法

在 MoviePilot 插件市场中添加仓库地址：
```
https://github.com/你的用户名/moviepilot-trailer-plugin
```

## 配置说明

| 配置项 | 说明 |
|--------|------|
| 预告片语言 | 中文优先 / 英文优先 / 不限制 |
| 下载来源 | YouTube / TMDb |
| 代理地址 | 可选，留空使用系统代理 |
| 监控路径 | 指定扫描的文件夹，多个用逗号分隔 |
| 定时全库扫描 | 每天指定时间自动扫描 |
| 跳过已有 | 避免重复下载 |

## 依赖

- yt-dlp（需在 MoviePilot 容器中安装）

## 兼容性

- 极影视：预告片显示为"播放源"，可用文件名/大小区分
- Jellyfin/VidHub：原生支持预告片播放
